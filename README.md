使用方法
白金牌电脑
1.先创建这个库的代码空间
2.先行【sudo su】（获得root权限）
3.安装宝塔面板
4.最后运行【rm -f /www/server/panel/data/admin_path.pl】（删除面板安全入口）
5.放行端口
